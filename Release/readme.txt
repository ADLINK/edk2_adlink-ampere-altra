This package
Files:
  BoardVersion.efi: tool reading board version.
  CapsuleApp.efi: capsule update tool.
  ComHpcAlt_tianocore_atf_*.cap : BIOS Capsule.
  ComHpcAlt_tianocore_atf_*.bin : 32MB BIOS binary for ROM programmer.
  ComHpcAlt_tianocore_atf_*.img : image for ROM programmer start at 0x400000.
  ComHpcAlt_scp_*.cap: SCP image
  Capsule.nsh: Update script board after A1 stepping.
  FwUi.efi: reboot to BIOS setup tool.
  readme.txt: this file.
  ReleaseNote.txt: Release notes

Usage:
  Run capsule.nsh in UEFI shell for on-board update.
